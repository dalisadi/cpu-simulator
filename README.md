# CPU Simulator (C)

## Description

This project is a CPU simulator written in C.
It is divided into multiple exercises (exo1.c → exo8.c), where each file adds new functionality.

---

## Files

* **exo1.c** → HashMap
* **exo2.c** → Memory
* **exo3.c** → Parser
* **exo4.c** → CPU
* **exo5.c** → Addressing
* **exo6.c** → Execution
* **exo7.c** → CPU Core
* **exo8.c** → Advanced Memory
